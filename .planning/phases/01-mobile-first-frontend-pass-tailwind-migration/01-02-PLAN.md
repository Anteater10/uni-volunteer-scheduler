  <files>frontend/src/lib/useFocusTrap.js</files>
  <read_first>frontend/src/lib</read_first>
  <action>Create frontend/src/lib/useFocusTrap.js. The hook accepts a ref to a container and an `active` boolean. While active:
1. On mount, store `document.activeElement` as `previouslyFocused`.
2. Query all focusable elements inside the container: `'a[href], button:not([disabled]), input:not([disabled]), select:not([disabled]), textarea:not([disabled]), [tabindex]:not([tabindex="-1"])'`.
3. Focus the first focusable element.
4. Add a `keydown` listener: on Tab without shift, if active === last focusable, prevent default and focus first. On Shift+Tab, if active === first, prevent default and focus last. On Escape, dispatch a custom event `focustrap-escape` on the container.
5. On unmount, remove listener and call `previouslyFocused?.focus?.()`.

Export as named `useFocusTrap`.</action>
  <verify>test -f frontend/src/lib/useFocusTrap.js && grep -q "export function useFocusTrap" frontend/src/lib/useFocusTrap.js && grep -q "Escape" frontend/src/lib/useFocusTrap.js</verify>
  <acceptance_criteria>
    - frontend/src/lib/useFocusTrap.js exists
    - Exports `useFocusTrap`
    - Implementation handles Tab, Shift+Tab, Escape
    - Restores focus on unmount via previouslyFocused
  </acceptance_criteria>
