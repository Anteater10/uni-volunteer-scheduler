**Structure:**
```jsx
import React, { useState, useMemo } from "react";
```

**State variables:**
- `drawerTemplate` (null | template object) — which template is being edited
- `createOpen` (boolean) — SideDrawer in create mode
- `showArchived` (boolean) — toggle to include archived templates
- `search` (string) — filter by name
- `typeFilter` ("all" | "seminar" | "orientation" | "module") — filter by type
- `page` / `PAGE_SIZE=10` — pagination
- `archiveConfirm` (null | slug) — archive confirmation modal target

**Query:**
```jsx
const listQ = useQuery({
  queryKey: ["adminTemplates", { include_archived: showArchived }],
  queryFn: () => api.admin.templates.list(showArchived ? { include_archived: true } : undefined),
});
```

**Mutations (4):**
- `createM` — api.admin.templates.create, onSuccess: close drawer, invalidate query, toast "Template created"
- `updateM` — api.admin.templates.update, onSuccess: close drawer, invalidate, toast "Template updated"
- `archiveM` — api.admin.templates.delete, onSuccess: close archiveConfirm, invalidate, toast "Template archived"
- `restoreM` — api.admin.templates.restore, onSuccess: invalidate, toast "Template restored"

**Filtering + pagination (client-side, useMemo):**
```jsx
const filtered = useMemo(() => {
  let result = listQ.data || [];
  if (search) result = result.filter(t => t.name.toLowerCase().includes(search.toLowerCase()));
  if (typeFilter !== "all") result = result.filter(t => t.type === typeFilter);
  return result;
}, [listQ.data, search, typeFilter]);

const totalPages = Math.ceil(filtered.length / PAGE_SIZE);
const pageData = filtered.slice((page - 1) * PAGE_SIZE, page * PAGE_SIZE);
```
